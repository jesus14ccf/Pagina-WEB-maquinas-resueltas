RESOLUCION DE MAQUINAS HTB

Maquina Nunchucks

En esta maáquina vamos a explotar una vulnerabilidad serser site template injection sobre un servidor web, es una maquina de hack the box catalogada como facil, aunque para alguien como yo que esta empezando es algo mas compleja sobre todo la escalada de privilegios al usuario root. Utilizo herramientas como burpsuite para interceptar peticiones, nmap para hacer escaneos, algo de fuerza bruta con wfuzz y algunas cosillas mas.

Maquina Cap

En esta máquina estaremos analizando un servicio web expuesto en el puerto 80, el cual presenta varias vulnerabilidades que explotaremos a lo largo del video. También estaremos trabajando con los servicios FTP (File Transfer Protocol) y SSH (Secure Shell) para obtener acceso a la máquina víctima. Utilizaremos TShark, la versión en consola de Wireshark, para analizar tráfico de red y capturar paquetes TCP provenientes del sistema objetivo. Finalmente, una vez dentro del sistema, realizaremos una escalada de privilegios hasta obtener acceso como root, abusando de una Linux capability asociada al binario Python.
